Stack Tecnológico del Proyecto

Esta demo es una version web ligera. El software final LatiNTax Audit Pro (.EXE) ha sido construido utilizando tecnologías robustas para garantizar velocidad, seguridad y portabilidad.

Backend 🐍
Python
Motor principal del software. Se encarga de la lógica de cálculo fiscal, procesamiento de XML y automatización de auditorías complejas.
Database 🗄️
SQLite
Base de datos integrada ligera. Permite almacenar millones de registros fiscales localmente sin necesidad de servidores externos, garantizando privacidad total.
Frontend/UI ⚛️
React + JS
Interfaz de usuario moderna y reactiva. React gestionó el diseño de los componentes visuales en fases de prototipado, integrado con lógica JS pura en el core.
Design 🎨
Figma
Herramienta de diseño UX/UI utilizada para crear los sistemas de diseño, prototipos de interacción y paleta de colores corporativa.
Build Tool 📦
PyInstaller
Clave para la distribución: Utilizamos PyInstaller para empaquetar todo el ecosistema Python, librerías y assets en un único archivo ejecutable (.EXE) para Windows. Esto permite a los usuarios instalar el software sin configurar entornos Python ni dependencias complejas.
